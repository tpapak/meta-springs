Published IPD example — logistic GLMM with spring-network view

Dataset: multinma::plaque_psoriasis_ipd (Phillippo, Dias, Ades et al., 2023 — plaque psoriasis IPD-MA following Riley-style individual-participant-data methodology).
n = 4114 patients, J = 4 studies, treatments: PBO, ETN, IXE_Q2W, IXE_Q4W, UST.
Outcome: pasi75 — Bernoulli (1 if 75% skin clearance at endpoint).
Model: pasi75 ~ trtc + wt_c + (1|studyc), fit by lme4::glmer (Laplace logistic GLMM).

Each patient spring pulls η_i toward y_i with hardness w_i (different per patient!
this is what distinguishes GLMM from LMM — hardnesses are not uniform).
Each study spring pulls u_j toward 0 with the same hardness 1/τ².
